# Room Management

Room types are managed in the **PMS** (Property Management System). The booking engine automatically pulls room data from the PMS.

## Adding a room type

1. Go to **[pms.vayada.com](https://pms.vayada.com)**
2. Navigate to **Room Types** in the sidebar
3. Click **Add Room Type**
4. Fill in:
   - **Name** — e.g. "Deluxe Double", "Family Suite"
   - **Description** — what makes this room special
   - **Max guests** — maximum occupancy
   - **Base price** — default nightly rate
   - **Amenities** — WiFi, AC, minibar, etc.
   - **Features** — balcony, sea view, etc.
   - **Photos** — upload room images
5. Save

The room type will automatically appear on the hotel's booking page.

## Editing a room type

1. Go to **Room Types** in the PMS
2. Click on the room type to edit
3. Make changes and save
4. Changes reflect on the booking page immediately

## Room availability

Room availability is managed through the **PMS Calendar**:

- Block dates when rooms are unavailable
- Set seasonal pricing (different rates for different periods)
- View and manage bookings on the calendar

## Book-direct benefits

In the **Booking Admin**, hotels can configure "book direct" benefits per room type — these are perks shown to guests who book through the hotel's direct booking page (e.g. "Free breakfast", "Late checkout", "10% off").

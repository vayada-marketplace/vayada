# Invite Codes

Invite codes let us pre-configure a hotel's setup before they even sign up. When a hotel uses an invite code during registration, their property, branding, rooms, and policies are automatically filled in.

## Creating an invite code

1. Go to **[admin.vayada.com](https://admin.vayada.com)** (Marketplace Admin)
2. Navigate to **Invite Codes** in the sidebar
3. Click **Create Invite Code**
4. Fill in the hotel's details through the setup wizard:
   - **Property info** — name, address, description, contact details
   - **Branding** — logo, colors, hero image
   - **Rooms** — room types, pricing, amenities
   - **Policies** — check-in/out times, cancellation policy
   - **Billing terms** — commission rate or fixed fee
5. Save — you'll get an invite code (e.g. `ABC123`)

## Sharing the invite code

Send the invite code to the hotel and ask them to:

1. Go to the Booking Admin signup page
2. Enter the invite code during registration
3. Their account will be created with everything pre-filled

## Managing invite codes

In the Marketplace Admin, you can:

- **View** all existing invite codes and their status (used/unused)
- **Edit** an unused invite code if details need to change
- **Delete** unused invite codes

:::note
Once an invite code has been used, it cannot be reused or edited. Create a new one if needed.
:::
